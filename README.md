# Trader Behavior Analysis Based on Market Sentiment

This project analyzes how crypto trader performance changes under different market sentiments such as Fear, Greed, Extreme Greed, and Neutral.

## What is done
- Cleaned and aligned market sentiment and trader datasets
- Calculated daily trader metrics like PnL, win rate, and trade count
- Merged trader behavior with daily market sentiment
- Compared performance across sentiment regimes
- Segmented traders into frequent/infrequent and consistent/inconsistent groups
- Visualized key insights

## Tools
Python, pandas, matplotlib, Jupyter Notebook

## How to run
Open `analysis.ipynb` and run all cells sequentially.
